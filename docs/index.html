How To Use Verba

A complete setup guide, model tuning instructions, and step-by-step feature walkthroughs.

⚙️ Setting Up the Downloaded Application

Step 1

Install & Launch

Download the installer for your OS (Windows .msi/.exe or Mac .dmg) and complete setup. Verba launches and stays ready in your system tray or menu bar.

Step 2

Trigger Hotkey Anywhere

Highlight text in any application and press Ctrl + Alt + P (or Cmd + Alt + P on Mac). Verba captures text instantly without leaving your active app.

Step 3

Select Style & Auto-Paste

Click any style preset or type custom instructions. Verba generates refined text using your selected AI model and automatically pastes it back.

💡

Model Setup, Accuracy & Privacy Disclaimers

1. Built-in Tiny Model (Default Offline Engine)

Verba comes pre-bundled with a lightweight local model running 100% offline on your device with no API keys needed.

Accuracy Note: Due to the small size of the built-in tiny model, accuracy might be quite limited. The primary use of the tiny model is for generative purposes (drafting text, brainstorming), while it may also work to polish simple text.
2. Upgrading to Llama 3B Q4 Model (For Better Local Accuracy)

If higher accuracy is required locally without relying on online servers:

  1. Download a Llama 3B Q4 model GGUF file (such as Llama 3.2 3B Instruct Q4_K_M).
  2. Place the downloaded .gguf file inside Verba's default models folder:
    Windows: %APPDATA%\com.verba.dev\models\
    macOS: ~/Library/Application Support/com.verba.dev/models/
  3. Rename the file name to exactly: llama-3.2-1b-instruct-q4_k_m.gguf
Speed Note: Larger size models require more system resources and will be slower to run than the tiny model.
3. Online Cloud API Providers & Privacy Guarantee

For maximum speed and state-of-the-art accuracy, you can switch to online API providers (OpenAI, Anthropic, Gemini, Groq, Ollama, or OpenRouter) in Settings → API Keys.

Provider Data Terms: When using online cloud APIs, the provider may process and use the data to train their future models depending on their terms of service.

Verba Privacy Guarantee: Verba does not retain any information anywhere online. All requests are sent directly from your client machine to your selected provider end-to-end.

Step-by-Step Guide for Every Feature

01

Global Hotkey Text Capture

Select text anywhere on your screen and press Ctrl+Alt+P (or Cmd+Alt+P on Mac). Verba pops up directly over your cursor with captured text pre-loaded.

02

8 Preset Polishing Styles

Instant one-click presets for common writing needs:

  • Concise: Streamlines text into brief, impactful phrasing.
  • Professional: Polishes tone for work, clients, and emails.
  • Detailed: Elaborates context with comprehensive detail.
  • Formal: Elevates vocabulary and academic structure.
  • Grammar: Corrects spelling, grammar, and punctuation.
  • Medical: Formats clinical reports and medical notes.
  • Summarize: Condenses long paragraphs into key takeaways.
  • Generative: Creative generation from custom prompts.
03

Custom Prompts & Number Shortcuts

Enter custom instructions into the prompt bar (e.g. "Rephrase as bullet points") or press keys 1 through 8 for instant shortcut execution.

04

Auto-Paste & Clipboard Restoration

Verba copies the polished output to your clipboard and injects it straight back into your focused text area. It restores your prior clipboard content so you don't lose copied data.

05

Multi-Provider AI Switching

Choose between 100% offline local models (Llama sidecar / local Ollama) or active online APIs (OpenAI GPT-4o, Claude 3.5, Gemini 1.5, Groq, OpenRouter) inside Settings.

06

Custom System Prompts & Keybindings

Customize system prompts for each style or rebind custom hotkeys under Settings → Preferences to match your personal workflow.

07

Local History Log

Review past text transformations, original inputs, models used, and execution timestamps stored safely on your machine in Settings → Local History.

08

Tray Engine & Automatic Updates

Verba operates silently in your system tray without interrupting desktop usage. Check for updates seamlessly in Settings → Updates.
